# Soul Sense EQ Test

The Soul Sense EQ Test is a desktop-based application built using Python's **Tkinter GUI** and **SQLite**. It offers an interactive Emotional Intelligence (EQ) assessment with 20 self-reflective questions and provides a total EQ score with an interpretation.
